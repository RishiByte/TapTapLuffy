## About the idea

**TapTapLuffy** is inspired by the viral **Bongo Cat** meme, reimagined with *Monkey D. Luffy* from **One Piece**.  
Each key press triggers a sound while Luffy reacts in real time, creating a playful and satisfying musical experience.
